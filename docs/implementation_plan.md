# はらどぅゲームス 公式サイト 実装計画

YouTuber「はらどぅゲームス」様向けの、高品質でモダンな公式サイトを作成します。

## 背景と目的
YouTuberのブランディングを高め、ファンにとって使いやすく魅力的なポータルサイトを構築します。最新の動画や人気動画、各種SNS活動への動線を一本化し、ダイナミックでリッチなデザイン（ゲーミングスタイル）を採用します。

## 使用技術とフレームワーク
- **コア**: Vite + React (TypeScript) - 高速な開発と描画パフォーマンスのため。
- **スタイリング**: Vanilla CSS - 柔軟で詳細なデザイン（Glassmorphismやネオンエフェクトなど）をカスタマイズするため。
- **アイコン**: `lucide-react` - 美しくモダンなUI表現のため。

## デザインシステム（Aesthetics）
- **テーマ**: プレミアム・ダーク・ゲーミング。深いネイビーや黒を基調とし、シアンやパープルなどの鮮やかなネオンカラーをアクセントに使用します。
- **UIエフェクト**: カードコンポーネント等へのGlassmorphism（半透明ガラス効果）、ホバー時のグロー（発光）エフェクトやスムーズな浮き上がりアニメーション。
- **タイポグラフィ**: 視認性が高くモダンなサンセリフ体（Google Fontsの Inter や Noto Sans JP など）を使用。

## 実装ステップとファイル構成

1. **プロジェクト初期化**:
   - `npm create vite@latest . -- --template react-ts` で現在のディレクトリに展開。
2. **ルーティング不要のSPA構成**:
   - トップページ単一（LP形式）で全てのコンテンツをスクロール閲覧できるように構成します（ユーザー体験の向上）。
3. **主要セクション (Components)**:
   - `Header`: サイト上部に固定。スクロール位置で背景の透明度が変化。
   - `Hero`: ファーストビュー。ダイナミックな背景とチャンネル名、YouTubeへの強力なCTA（Call to action）ボタン。
   - `LatestVideos`: 最新動画を大きくフィーチャー。
   - `PopularVideos`: 人気の動画をグリッドレイアウトで配置。画像ホバーで再生アイコンが浮かび上がるリッチなインタラクション。
   - `SocialLinks`: Twitter（X）や他プラットフォームへのリンク集。
4. **SEO最適化**:
   - `index.html` の `title`, `meta description` などを適切に設定。

## 検証・テスト計画 (Verification Plan)

### Automated/Local Tests
- `npm run dev` を実行し、ローカルサーバーで以下の点を確認します。
  1. エラーなくビルド・起動できること。
  2. 各種コンポーネントが正しくレンダリングされていること。

### Manual Verification
- **ブラウザプレビュー**:
  - Chrome等で `http://localhost:5173` を開き、目視で確認。
  - 開発者ツールのデバイスツールバーを使用し、モバイル（SP）サイズでのレスポンシブデザイン崩れがないか確認。
  - ボタンのホバーエフェクトやスクロールアニメーションが滑らかに動作するかテスト。
